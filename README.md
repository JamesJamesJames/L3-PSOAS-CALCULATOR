# L3 Psoas Index (Jones Method) Calculator

A lightweight single-page calculator for estimating total psoas area at L3 and normalized psoas index (mm²/m²), with sex-specific threshold classification.

## Run locally

No build step is required.

1. Open `index.html` directly in a browser, **or**
2. Serve the folder with a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
